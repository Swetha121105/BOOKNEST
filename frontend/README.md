# 📚 BookNest - Event Management System

BookNest is a full-stack MERN application designed to manage and discover events. It features user authentication, event creation, and a real-time connection to a MongoDB database.

## 🚀 Features
- **User Authentication:** Secure signup and login using JWT (JSON Web Tokens).
- **Event Management:** Create, view, and filter events seamlessly.
- **Responsive UI:** Built with React for a smooth user experience.
- **Backend API:** Robust Node.js and Express server with MongoDB integration.

## 🛠️ Tech Stack
- **Frontend:** React.js, Axios, CSS/Tailwind
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Local/Atlas)
- **Authentication:** JWT (jsonwebtoken)
